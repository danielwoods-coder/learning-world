简介

HTML（超文本标记语言）是用于创建网页的标准标记语言。使用 HTML 你可以构建自己的网站。

HTML 从语义上描述网页的结构，最初也包含了对文档外观的提示。

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>页面标题</title>
</head>
<body>
    <h1>你好世界</h1>
</body>
</html>

CSS（层叠样式表）用于为网页添加样式和布局。例如，你可以使用 CSS 控制字体、颜色、间距以及 HTML 元素的整体视觉呈现。

HTML 元素

HTML 元素由开始标签、内容和结束标签组成。HTML 元素是网页的构建块。

<标签名>这里是内容...</标签名>

以下是一些常见的 HTML 元素：

  • <header> - 表示介绍性内容，通常是一组介绍性或导航性辅助内容
  • <nav> - 表示文档中提供导航链接的部分
  • <main> - 指定文档的主要内容
  • <section> - 定义文档中的章节
  • <p> - 定义段落
  • <code> - 以等宽字体显示内容，用于内联代码片段
HTML 属性

属性为 HTML 元素提供额外信息。它们总是在开始标签中指定，以名称/值对的形式出现。

<a href="https://example.com">点击这里</a>

常用属性包括：

  • href - 指定链接指向的页面 URL
  • src - 指定图像的路径
  • alt - 为图像指定替代文本
  • class - 为元素指定一个或多个类名
  • id - 为元素指定唯一的 id
  • style - 为元素指定内联 CSS 样式
CSS 基础

CSS 用于为 HTML 元素添加样式。CSS 规则由选择器和声明块组成。

选择器 {
    属性: 值;
}

CSS 选择器用于"查找"（或选择）你要设置样式的 HTML 元素。常用选择器包括：

  • 元素 - 按标签名选择所有元素
  • #id - 按 id 属性选择元素
  • .class - 选择具有特定类的所有元素

CSS 盒模型：网页设计中的每个元素都是一个矩形盒子，具有外边距（margin）、边框（border）、内边距（padding）和内容（content）属性。

最佳实践

编写 HTML 和 CSS 时请遵循以下最佳实践：

  1. 始终声明正确的 DOCTYPE 以确保标准兼容渲染
  2. 使用语义化 HTML 元素（<nav>、<main>、<article>）以提高可访问性和 SEO
  3. 将 CSS 保持在单独的文件中，而非内联样式，以提高可维护性
  4. 为图像使用有意义的 alt 文本来提高可访问性
  5. 使用 W3C 验证器验证你的 HTML
  6. 使用响应式设计技术确保你的网站在所有设备上都能正常工作
