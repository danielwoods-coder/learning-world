# 代码世界

此仓库是一个用于 HTML、CSS、JavaScript 和 Python 项目的学习练习场。

## 目录结构

- `HTML/` — 30 多个独立的前端练习，包括表单、布局、动画和响应式页面。
- `JavaScript/` — 50 多个练习和小型应用，包含算法、交互式 UI 项目和实用工具。
- `Python/` — 10 多个算法与数据结构练习，以及一些小工具。

## 使用说明

### HTML
在浏览器中打开任意 `.html` 文件，或使用 Live Server 扩展获得更好的开发体验。

### JavaScript
在 `JavaScript/` 文件夹中使用 Node.js 运行独立脚本。

```powershell
cd "d:\代码世界\Learning-World\JavaScript"
node <filename>.js
```

对于基于 Web 的 JavaScript 项目，请在浏览器中打开该文件夹下的 `index.html` 或 `*.html`。

### Python
在 `Python/` 文件夹中运行 Python 脚本。

```powershell
cd "d:\代码世界\Learning-World\Python"
python <filename>.py
```

## 约定

- 文件名使用描述性名称，并使用连字符，例如 `cafe-menu.html`。
- 项目大多数是独立练习，HTML/CSS/JS 文件通常使用相同的基础名称配对。
- 不需要统一构建系统。
