# Agent Instructions for Learning World

This workspace is a comprehensive learning repository containing HTML/CSS, JavaScript, and Python projects for practicing programming skills.

## Project Structure
- **HTML/**: 32+ projects for front-end design, forms, layouts, and CSS animations.
- **JavaScript/**: 50+ projects including algorithms, interactive web apps, and utilities.
- **Python/**: 11 projects focused on algorithms, data structures, and OOP.

See [README.md](README.md) for detailed project categories and examples.

## Key Conventions
- Projects are standalone exercises; no unified build system.
- File naming: descriptive and hyphenated (e.g., `cafe-menu.html`).
- Paired files share base names for HTML/CSS/JS combinations.
- Linting: Use ESLint for JavaScript files (flat config with plugins for JS/TS/React, JSON, Markdown, CSS).

## Running Projects
- HTML/CSS: Open in browser or use live server.
- JavaScript: Run with `node filename.js`; lint with `npx eslint .` in JavaScript/ folder. For apps like `app.js`, use Node.js server.
- Python: Run with `python filename.py`.

## Maintenance
When adding new projects, update [README.md](README.md) with the project name and description in the appropriate category.
